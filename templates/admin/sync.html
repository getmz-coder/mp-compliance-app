Programación MP

Excel con los equipos del ciclo de mantenimiento preventivo. Cada carga actualiza o agrega registros en base al consecutivo.

Haz clic para seleccionar o arrastra el archivo aquí

Columnas requeridas: fecha_programacion · consecutivo · categoria · estado_vehiculo · vehiculo · familia · rutina · desviacion · indice_desviacion · estado_mp

Maestro Filtración Opcional

Catálogo de filtros por equipo. Solo es necesario subir cuando el catálogo cambie. Reemplaza completamente la tabla existente.

Haz clic para seleccionar o arrastra el archivo aquí

Columnas requeridas: EQUIPO · TIPO · NOMBRE ARTÍCULO · CODIGO SAP · TIPO FILTRO

Homólogos de Filtros Opcional

Catálogo de equivalencias entre códigos SAP. Reemplaza completamente la tabla existente al subir un nuevo archivo.

Haz clic para seleccionar o arrastra el archivo aquí

Columnas requeridas: Grupo · Tamaño Grupo · Estado · Codigo SAP · Descripcion

Frecuencias de Rutinas Opcional

Tabla de frecuencias por rutina MP (horas o km por ciclo, días entre mantenimientos). Usada por el módulo de Planeación Predictiva.

Haz clic para seleccionar o arrastra el archivo aquí

Hoja requerida: DB_FRECUENCIAS · Columnas: rutina · frecuencia_medidor · frecuencia_dias
